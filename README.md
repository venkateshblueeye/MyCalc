📂 Step 1: Download the ZIP
On the GitHub page, click the green Code button.
Select Download ZIP.
Save it to a convenient location on your computer.

📦 Step 2: Extract the ZIP
Right-click the downloaded file → Extract All (Windows) or use Finder/Archive Utility (Mac).
This will create a folder containing the project files.

🛠 Step 3: Import into Android Studio
Open Android Studio.
Go to File → New → Import Project.
Navigate to the extracted folder and select it.
Android Studio will sync Gradle and set up the environment.

✅ Step 4: Verify the Project

Check the Gradle sync status at the bottom of Android Studio.
Ensure there are no errors in the Build Output window.
Open the MainActivity.java or MainActivity.kt file to confirm the entry point.
Run the app on an emulator or connected device to verify functionality.
